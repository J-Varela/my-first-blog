14. You are creating an application that references the Azure OpenAI REST API for a DALL-E model.

You plan to use thumbnails of the images that DALL-E generates and display them in a table on webpage.

You need to fint the image URLs in the JSON response.

**Which element should you review**?

- [x] **the result element**. The result from the initial requests does not immediately return the results of the image generation process. Instead, the response includes an **operation-location** header with a URL for a calback service that your application code can poll until the results of the image generation are ready. The **result** element includes a collection of **url** elements, each of which references a PNG image file generated from the prompt.

Adding this information allows the model to reference both specific data probided and its pretrained knowledge to provide more effective responses. Azure OpenAI on your data utilizes the search ability of Azure AI Search to add the relevant data chunks to a prompt.

32. **You are building a model that uses Conversation Language Understanding (CLU)**

**Which metric should you use?**

- [x] precision.

  - explanation: Precision measures how precise/accurate a model is. It is the ratio between the correctly identified positives (true positive) and all identified positive. The precision metric reveals how many of the predicted classes are correctly labeled.

  Recall measures the model's ability to predict actual positive classes.

  F1 score is a function of precision an recall.

  BLEU is from the Azure AI Translator service.
