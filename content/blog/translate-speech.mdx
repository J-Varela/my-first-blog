1. Which SDK object should you use to specify the language into which you want speech translated?

- [x] SpeechTranslationConfig. Specify target languages in the SpeechTranslationConfig Object.

2. Which SDK object should you use as a proxy for the Translation API of the Azure AI Speech service?

- [x] Use a TranslationRecognizer to call the Translation API of the Azure AI Speech service.

3. When translating speech, in which cases can you use the Synthesizing event to synthesize the translation and speech?

- [x] You can only use event-based synthesis when translating to a single taget language.
